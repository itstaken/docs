*java.awt.Window.Type_Description*

Enumeration of available window types. 

A window type defines the generic visual appearance and behavior of a top-level 
window. For example, the type may affect the kind of decorations of a 
decoratedFrameorDialoginstance. 

Some platforms may not fully support a certain window type. Depending on the 
level of support, some properties of the window type may be disobeyed.
