*android.test.suitebuilder.annotation.Suppress_Description*

Use this annotation on test classes or test methods that should not be included 
in a test suite. If the annotation appears on the class then no tests in that 
class will be included. If the annotation appears only on a test method then 
only that method will be excluded.
