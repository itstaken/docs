*android.util.LayoutDirection_Description*

A class for defining layout directions. A layout direction can be left-to-right 
(LTR) or right-to-left (RTL). It can also be inherited (from a parent) or 
deduced from the default language script of a locale. 



*int_android.util.LayoutDirection.INHERIT*

Horizontal layout direction is inherited. 


*int_android.util.LayoutDirection.LOCALE*

Horizontal layout direction is deduced from the default language script for the 
locale. 


*int_android.util.LayoutDirection.LTR*

Horizontal layout direction is from Left to Right. 


*int_android.util.LayoutDirection.RTL*

Horizontal layout direction is from Right to Left.
