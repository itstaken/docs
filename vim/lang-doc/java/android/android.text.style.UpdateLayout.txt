*android.text.style.UpdateLayout_Description*

The classes that affect character-level text formatting in a way that triggers 
a text layout update when one is added or removed must implement this 
interface. This interface also includes (|android.text.style.UpdateAppearance|) 
since such a change implicitly also impacts the appearance.
