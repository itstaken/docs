*android.view.HapticFeedbackConstants_Description*

Constants to be used to perform haptic feedback effects via 
(|android.view.View|) 



*int_android.view.HapticFeedbackConstants.FLAG_IGNORE_GLOBAL_SETTING*

Flag for View.performHapticFeedback(int, int)(|android.view.View|) : Ignore the 
global setting for whether to perform haptic feedback, do it always. 


*int_android.view.HapticFeedbackConstants.FLAG_IGNORE_VIEW_SETTING*

Flag for View.performHapticFeedback(int, int)(|android.view.View|) : Ignore the 
setting in the view for whether to perform haptic feedback, do it always. 


*int_android.view.HapticFeedbackConstants.KEYBOARD_TAP*

The user has pressed a soft keyboard key. 


*int_android.view.HapticFeedbackConstants.LONG_PRESS*

The user has performed a long press on an object that is resulting in an action 
being performed. 


*int_android.view.HapticFeedbackConstants.SAFE_MODE_DISABLED*

This is a private constant. Feel free to renumber as desired. 


*int_android.view.HapticFeedbackConstants.SAFE_MODE_ENABLED*

This is a private constant. Feel free to renumber as desired. 


*int_android.view.HapticFeedbackConstants.VIRTUAL_KEY*

The user has pressed on a virtual on-screen key.
