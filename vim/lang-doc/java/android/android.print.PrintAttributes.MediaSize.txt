*android.print.PrintAttributes.MediaSize_Description*

This class specifies a supported media size. Media size is the dimension of the 
media on which the content is printed. For example, the 
(|android.print.PrintAttributes.MediaSize|) media size designates a page with 
size 8.5" x 11".
