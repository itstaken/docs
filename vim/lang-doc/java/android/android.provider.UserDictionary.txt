*android.provider.UserDictionary_Description*

A provider of user defined words for input methods to use for predictive text 
input. Applications and input methods may add words into the dictionary. Words 
can have associated frequency information and locale information. 



*java.lang.String_android.provider.UserDictionary.AUTHORITY*
