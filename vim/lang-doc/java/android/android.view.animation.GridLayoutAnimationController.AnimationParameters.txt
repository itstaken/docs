*android.view.animation.GridLayoutAnimationController.AnimationParameters_Description*

The set of parameters that has to be attached to each view contained in the 
view group animated by the grid layout animation controller. These parameters 
are used to compute the start time of each individual view's animation. 



*int_android.view.animation.GridLayoutAnimationController.AnimationParameters.column*

The view group's column to which the view belongs. 


*int_android.view.animation.GridLayoutAnimationController.AnimationParameters.columnsCount*

The number of columns in the view's enclosing grid layout. 


*int_android.view.animation.GridLayoutAnimationController.AnimationParameters.row*

The view group's row to which the view belongs. 


*int_android.view.animation.GridLayoutAnimationController.AnimationParameters.rowsCount*

The number of rows in the view's enclosing grid layout.
