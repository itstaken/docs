*android.support.v8.renderscript.Script.KernelID_Description*

KernelID is an identifier for a Script + root function pair. It is used as an 
identifier for ScriptGroup creation. 

This class should not be directly created. Instead use the method in the 
reflected or intrinsic code "getKernelID_funcname()".
