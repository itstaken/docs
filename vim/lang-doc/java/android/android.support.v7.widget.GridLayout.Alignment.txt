*android.support.v7.widget.GridLayout.Alignment_Description*

Alignments specify where a view should be placed within a cell group and what 
size it should be.

rowSpec(|android.support.v7.widget.GridLayout.LayoutParams|) and a 
columnSpec(|android.support.v7.widget.GridLayout.LayoutParams|) each of which 
contains analignment. Overall placement of the view in the cell group is 
specified by the two alignments which act along each axis independently.
