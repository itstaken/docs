*android.content.ClipboardManager.OnPrimaryClipChangedListener_Description*

Defines a listener callback that is invoked when the primary clip on the 
clipboard changes. Objects that want to register a listener call 
addPrimaryClipChangedListener()(|android.content.ClipboardManager|) with an 
object that implements OnPrimaryClipChangedListener.
