*android.webkit.JavascriptInterface_Description*

Annotation that allows exposing methods to JavaScript. Starting from API level 
(|android.os.Build.VERSION_CODES|) and above, only methods explicitly marked 
with this annotation are available to the Javascript code. See 
(|android.webkit.WebView|) for more information about it.
