*android.provider.CalendarContract.CalendarSyncColumns_Description*

Generic columns for use by sync adapters. The specific functions of these 
columns are private to the sync adapter. Other clients of the API should not 
attempt to either read or write this column. These columns are editable as part 
of the Calendars Uri, but can only be read if accessed through any other Uri.
