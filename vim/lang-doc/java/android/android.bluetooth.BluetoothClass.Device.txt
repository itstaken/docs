*android.bluetooth.BluetoothClass.Device_Description*

Defines all device class constants. Each (|android.bluetooth.BluetoothClass|) 
encodes exactly one device class, with major and minor components. The 
constants in (|android.bluetooth.BluetoothClass.Device|) represent a 
combination of major and minor device components (the complete device class). 
The constants in (|android.bluetooth.BluetoothClass.Device.Major|) represent 
only major device classes. See (|android.bluetooth.BluetoothClass.Service|) for 
service class constants.
